## Installation

```ruby
$ git clone
$ bundle install
$ rails db:migrate
$ rails db:seed
$ rails s
$ npm start --prefix client
```

Open Chrome browser, and redirect to 'http://localhost:4000' to start the app.

**Alternatively, it is fully deployed on Heroku!**
